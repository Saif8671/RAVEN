## 🦅 Overview

**RAVEN** (Resilient Automated Vulnerability & Engineering Network) is a comprehensive cybersecurity platform designed specifically to bridge the security gap for small businesses. It combines automated scanning, real-time threat detection, and AI-driven incident response into a single, intuitive dashboard.

In an era where SMBs are primary targets for cyberattacks, RAVEN provides the tools traditionally reserved for enterprise SOCs in a accessible, "glassmorphism" inspired interface.

## ✨ Key Features

### 🛡️ Vulnerability Scanner
Automated assessment of your digital footprint. Identifies misconfigurations, outdated services, and potential entry points before attackers do.

### 📧 Email Security & Phishing Defense
Advanced analysis of email headers and content to detect sophisticated phishing attempts. Protects your team from business email compromise (BEC).

### 🔍 Breach & Data Leak Detection
Monitors the dark web and public data leaks for compromised company credentials and sensitive data exposure.

### 🎣 Phishing Simulation
Train your workforce with realistic, safe phishing simulations. Measure and improve your organization's human-firewall resilience.

### 🤖 AI Incident Response (Powered by GROQ)
A 24/7 intelligent assistant that provides immediate remediation steps during a security event, acting as your virtual Chief Information Security Officer (vCISO).

## 🛠️ Tech Stack

### Frontend
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS (Custom Glassmorphism UI)
- **Animations:** Framer Motion
- **Icons:** Lucide React

### Backend
- **Runtime:** Node.js (ES Modules)
- **Framework:** Express.js
- **Security:** Helmet.js, CORS, Zxcvbn
- **AI Integration:** Groq SDK
- **Communication:** Nodemailer, Axios

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Saif8671/RAVEN.git
   cd RAVEN
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Create a .env file based on .env.example
   npm run start
   ```

3. **Frontend Setup**
   ```bash
   cd ../raven-app-vite
   npm install
   npm run dev
   ```
